Mac における zshのPATH設定

May 25, 2014 at 03:00 PM | categories: tips | View Comments

Mac環境(tmux含む)のPATHがおかしくなったので、メモ

% cat /etc/zprofile
% cat /etc/zshenv
# system-wide environment settings for zsh(1)
if [ -x /usr/libexec/path_helper ]; then
   PATH=''
   eval `/usr/libexec/path_helper -s`
fi
source /etc/profile.d/rvm.sh
% cat ~/.zlogin
% cat ~/.zprofile
% cat ~/.zshrc
(略)
export PATH=$PATH:$HOME/go/1.1.2/bin
% echo $PATH
/usr/local/rvm/gems/ruby-1.9.3-p392/bin:
/usr/local/rvm/gems/ruby-1.9.3-p392@global/bin:
/usr/local/rvm/rubies/ruby-1.9.3-p392/bin:
/usr/local/rvm/bin:
/opt/local/bin:
/usr/bin:/bin:
/usr/sbin:
/sbin:
/usr/local/bin:
/Users/satoshi/go/1.1.2/bin
% ls /etc/paths.d/
% cat /etc/paths
