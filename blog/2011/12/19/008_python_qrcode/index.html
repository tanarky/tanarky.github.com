環境構築手順

libqrencodeのpython bindingによるQRコード生成

まずはinstall

% sudo aptitude install libqrencode3 libqrencode-dev
% pip install qrencode

動作確認

サンプルコード

import qrencode

e = Encoder()
image = e.encode('http://tanarky.com/', { 'width': 100 })
image.save("/tmp/qrcode.png")
